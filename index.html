GET Endpoints

GET /api/brat

Generate Brat themed images with customizable text and animation options.

Parameters
text
Text to display on the image (required)
string
isAnimated
Enable animation effect
boolean
delay
Animation delay in milliseconds
number
type
Response type: json or redirect
string
Example Request
GET /api/brat?text=Hello&isAnimated=true&delay=1000&type=json
Example Response
{ "success": true, "url": "data:image/png;base64,...", "result": { "url": "https://example.com/image.png" } }
GET /api/hd

Upscale images to HD quality using AI enhancement.

Parameters
url
Image URL to upscale (required)
string
Example Request
GET /api/hd?url=https://example.com/image.jpg
Example Response
{ "success": true, "result": { "upscaled_url": "https://example.com/upscaled.jpg" } }
GET /api/tiktok

Download TikTok videos without watermark.

Parameters
url
TikTok video URL (required)
string
mode
Response mode: json or redirect
string
Example Request
GET /api/tiktok?url=https://tiktok.com/@user/video/123456&mode=json
Example Response
{ "id": "123456", "caption": "Video title", "play": "https://example.com/video.mp4", "music": "https://example.com/music.mp3", "author": "username" }

Error Responses

All endpoints return consistent error responses when something goes wrong.

Error Response Format
{ "error": "Error message", "message": "Detailed error description", "raw": {} // Optional: Raw error data }
Common HTTP Status Codes
400 - Bad Request (Missing parameters) 405 - Method Not Allowed 429 - Rate Limited 500 - Internal Server Error
